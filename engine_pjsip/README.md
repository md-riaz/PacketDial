# PJSIP Engine

Place pjproject source code here:

- `engine_pjsip/pjproject/`

Options:
1) Add as a git submodule:
   - git submodule add <pjproject-repo-url> engine_pjsip/pjproject

2) Download pjproject release and extract into that folder.

Then follow `scripts/build_pjsip.ps1` to produce:
- `engine_pjsip/build/out/include`
- `engine_pjsip/build/out/lib`
